# セットアップ
## 1. git clone
```
$ https://github.com/APPLE4869/phrase-art-web.git && cd phrase-art-web
```

## 2. Dockerを起動
```
$ docker-compose up
```

## 3. データベースを用意
```
$ mysql -u root -h 127.0.0.1 -e 'create database phrase_art_development character set utf8mb4 collate utf8mb4_bin;'
```

## 4. マイグレーション
```
$ gradle build
```

## 5. サーバー起動
```
$ ./gradlew bootRun
```
